Retrieving data from Google Maps API / Waze API

By executing a script that retrieves data with a set interval it is possible to save Google Maps travel times taking in consideration traffic congestions in real-time. The datacollection can be done together with other analyses to add a new value

Data can be fetched for instance every 30 seconds and stored in a simple tab-separated *.txt file. It can also be written to a databasefile including other parameters from different sources
