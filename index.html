Example projects/project deliverables/methods by Anton van Berlekom

Click on the images to try / for more information. Most of the examples are based on Python 2.7/Python 3.X+ (open source) and free online libraries (Google APIs etc).

1. Google Charts

Creating interactive graphs with Google Charts.

  • Can be used to display fast interactive graphs (applyable filters and commands)
  • Easy customizable and extremely responsive (quick)

2. Google Maps Api v3

Creating interactive maps with Google Maps Api v3

  • Show markers, polylines, polygons in an interactive map
  • Small size document (often smaller than 1 MB)
  • Online dependancy

3. Sphinx

Creating reports for PDF and HTML in one sweep

  • Combine new interactive tools with PDF-format reports
  • Low amount of formatting required

4. Flask

Creating graphic user interface (GUI) with Flask

  • Powerful customization options and easy to understand UI
  • Flask is Python-powered and easily combined with Python-scripts

5. Google Maps Travel Time (API)

Retrieving data from Google Maps API and Waze API

  • Data can be retrieved with fixed intervals from different A to B coordinates
  • Can be used as a complementary data collection method

6. QGIS and webmaps addin

Creating maps with QGIS and OpenStreetMap

  • Quickly generates interactive maps in html-format that can be used together with other outputs
  • Displays *.shp files in an easy to understand and interactive format

7. Pandas

Analyzing big data with Pandas

  • Python-based library to analyze large amounts of data
  • Can be combined with other methods described in this page

8. Excel

Collect data with Microsoft Excel

  • An alternative way of collecting information.
  • Data is gathered with a macro-activated excel file

9. Combining things

Automating (combining things)

  • By combining the methods described above it is possible to combine collect data, crunch data, display data and report data.
